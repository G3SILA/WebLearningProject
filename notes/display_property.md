# Display Property

1. block element 
    - takes the entire line
    - no matter the width 
    - e.g. paragraph `<p>`

2. inline-block element
    - as much as space needs
    - e.g. image `<img>`

3. inline element 
    - within a line of text
    - e.g. `<strong>`

## display - switch between block and inline-block element
- block => inline-block
- `display: inline-block`     

- inline-block => block 
- `display: block`
